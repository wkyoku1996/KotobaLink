# Mock

Mock fixtures and demo-only datasets live here.

Use this directory for:
- page fixture data
- mock course, lesson, and message datasets
- lesson resource fixtures
- assessment question fixtures

Editing rule:
- If a value represents fake business data shown in the demo, edit it here.
- If a value is app-level configuration, edit `config/`.

Current files:
- `demo-source.js`: the main demo data source and demo assembly logic
