# Admin

React admin frontend for KotobaLink operations.

## Stack

- React
- TypeScript
- Vite

## Local development

```bash
pnpm install
pnpm --filter @kotobalink/admin dev
```

Default local URL:

- `http://localhost:5173`

Configured pages:

- `/`: dashboard
- `/courses`: course management scaffold
- `/orders`: order management scaffold
- `/settings`: settings scaffold
