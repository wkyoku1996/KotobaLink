# 共有 Hero Card

## 基本情報

| 項目 | 内容 |
| --- | --- |
| 控件 ID | `CMP-SHARED-001` |
| 控件名 | Hero Card |
| 種別 | 共有レイアウト控件 |
| 実装元 | `apps/miniapp/app.wxss` |

## 責務

Hero Card はページ冒頭の主見出し領域を統一するための共有外枠です。

責務:
- ページ冒頭の強調領域を提供する
- タイトル、サブタイトル、バッジ群を同じ視覚ルールで表示する
- 背景グラデーションと余白ルールを共通化する

責務外:
- 中身の文言内容
- ボタンの挙動
- データ取得

## スタイル仕様

| 項目 | 値 |
| --- | --- |
| 背景 | `linear-gradient(145deg, #1f6a62, #315f74)` |
| 角丸 | `28rpx` |
| パディング | `32rpx 28rpx` |
| 文字色 | `#fff8ef` |
| 影 | `0 16rpx 40rpx rgba(69, 53, 36, 0.08)` |

## 構成要素

- `hero-title`
- `hero-subtitle`
- `badge-row`
- ページ専用の追加ボタンや補足領域

## 入力項目

Hero Card 自体は prop を持ちません。中身の要素は各ページ WXML で直接定義します。

## 利用ページ

- 首页
- 课程
- 消息
- 我的
- 活动
- 课表
- 支付结果
- 课程详情
- lesson / assessment 详情

## 実 API 化時の注意

Hero Card は表示外枠のみを持つため、実 API 化の際にも shared layout として維持可能です。
