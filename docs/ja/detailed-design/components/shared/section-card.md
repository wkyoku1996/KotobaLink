## 責務

Section Card はページ内の各情報ブロックを統一する共有コンテナです。見出し、本文、一覧、ボタン群の外枠として使用されます。

## backend データ要求

この共有コンポーネント自体に直接対応する backend エンティティや API はありません。利用先コンポーネント側で定義したデータ要求を受けて表示枠として機能します。
