# 共有 Section Card

## 基本情報

| 項目 | 内容 |
| --- | --- |
| 控件 ID | `CMP-SHARED-002` |
| 控件名 | Section Card |
| 種別 | 共有コンテナ控件 |
| 実装元 | `apps/miniapp/app.wxss` |

## 責務

Section Card はページ本文内の各セクションを同じカード構造で包むための共有容器です。

責務:
- セクション単位の背景、余白、境界線を統一する
- `section-title` と `section-desc` の基本ルールを統一する
- 内部にページ専用控件を載せる器を提供する

責務外:
- セクション内部の業務ロジック
- ボタン処理
- データ状態判断

## 基本スタイル

| 項目 | 値 |
| --- | --- |
| 背景 | `rgba(255, 253, 248, 0.94)` |
| 境界線 | `1rpx solid rgba(108, 87, 63, 0.12)` |
| 角丸 | `28rpx` |
| マージン上 | `22rpx` |
| 内側余白 | `26rpx 24rpx` |

## 構造

- `section-title`
- `section-desc`
- ページ専用の本文ブロック
- 任意の `action-row`

## 主要利用箇所

首页では以下で使用します。

- 能力雷达图
- 近期课表
- 活动入口
- 今日任务

他ページでも一覧、統計、明細表示に広く使用します。
