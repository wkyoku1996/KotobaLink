# 会員 現在の特典

## 基本情報

| 項目 | 内容 |
| --- | --- |
| コンポーネント ID | `CMP-MEMBER-002` |
| 実装元 | `pages/account/membership/index.wxml` |

## 責務

現在の会員状態で利用できる课程服务权益と活动参与权益を固定表示します。現行 demo では動的切替はなく、読取専用の説明セクションです。

プラン購入やプラン比較の起点ではなく、現在の会員状態で前提として見せる説明領域として扱います。

## 表示フィールド表

| 表示名 | キー | 型 | 例 |
| --- | --- | --- | --- |
| 课程服务权益 | 固定文言 | `string` | `基础课程、学习档案、每日任务、消息提醒` |
| 活动参与权益 | 固定文言 | `string` | `可查看会员活动、优先报名学习沙龙` |
