# 受講中コース クラス基本情報

## 基本情報

| 項目 | 内容 |
| --- | --- |
| コンポーネント ID | `CMP-ENROLLED-003` |
| 実装元 | `pages/account/profile-course-detail/index.wxml` |

## 責務

受講中コースのクラス情報を表示します。クラス名、時間帯、受講形態など、受講枠に関係する属性を確認するためのセクションです。

## 表示フィールド表

| 表示名 | キー | 型 | 用途 |
| --- | --- | --- | --- |
| 項目ラベル | `item.label` | `string` | 行識別 |
| 項目値 | `item.value` | `string` | クラス属性表示 |
