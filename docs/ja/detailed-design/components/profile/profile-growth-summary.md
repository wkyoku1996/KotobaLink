# マイページ 成長サマリー

## 基本情報

| 項目 | 内容 |
| --- | --- |
| コンポーネント ID | `CMP-PROFILE-003` |
| 実装元 | `pages/tab/profile/index.wxml` |
| データ源 | `growth.summaryCards` |

## 責務

成长轨迹概要セクションとして、現在の主要指標カードを並列表示します。各カードは値、ラベル、補足説明で構成されます。

## 表示フィールド表

| 表示名 | キー | 型 |
| --- | --- | --- |
| 値 | `item.value` | `string` |
| ラベル | `item.label` | `string` |
| 補足説明 | `item.desc` | `string` |
