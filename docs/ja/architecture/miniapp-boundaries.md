# Miniapp 境界

## 文書目的

この文書は、どの内容を `apps/miniapp` に残し、どの内容を今後共有層へ移すかを整理するものです。

## `apps/miniapp` に残す内容

- `wx.*` を呼び出すロジック
- 画面遷移
- `Page(...)` ライフサイクル
- アクセシビリティ behavior とテンプレート連携
- ミニアプリ専用表示状態
- demo 用 glue code

## `packages/shared` に入る内容

- 日付処理
- スコア変換
- メッセージ並び替え
- 時間割並び替え
- プラットフォーム API に依存しない純ロジック

## `packages/types` に入る内容

- データエンティティ契約
- 列挙状態
- API payload 定義
- クエリパラメータ契約
